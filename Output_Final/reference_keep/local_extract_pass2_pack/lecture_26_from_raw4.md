# Лекція 26. ПРИНЦИП СУПЕРПОЗИЦІЇ — ОСНОВА ТЕОРІЇ ЛІНІЙНИХ СИСТЕМ

ЦЕЛЬ ЛЕКЦИИ: Познакомить читателя с принципом суперпозиции и показать, как с его помощью исходную задачу можно разложить на подзадачи, затем решить сразу все подзадачи и, сложив полученные результаты, найти решение исходной задачи. Мы хотим также показать, что два основных метода решения линейных уравнений—метод разделения переменных и метод интегральных преобразований—используют принцип суперпозиции.

Предположим, что инженеру нужно найти отклик линейной системы и на входное воздействие f. Общий подход к решению задачи таков:

- 1) разложить f на элементарные воздействия  $f = \sum f_k$ ;
- 2) найти отклик системы  $u_k$  на воздействие  $f_k$ ;

3) сложить отклики  $u = \sum_{i=1}^{n} u_{i}$ .

Если система линейная, то полученная сумма и является откликом системы на входное воздействие f. В этом состоит принцип суперпоэиции (см. рнс. 26.1).

РИС, 26.1. Основная идея принципа суперпоэнции; L—дифференциальный оператор; f—вход; u—выход.

Эту основную идею можно использовать при решении сложных смешанных задач, разбивая их на простые подзадачи, ре-

шая каждую из них и затем суммируя полученные результаты (конечно, дифференциальное уравнение и граничные условия должны быть линейны).

## Разложение смешанной задачи на две более простые

Предположим, нам необходимо решить линейную задачу (обозначим ее P)

(УЧП) 
$$

u_t = u_{xx} + \sin(\pi x), \quad 0 < x < 1, \quad 0 < t < \infty,

$$

(Р) (ГУ) 
$$

\begin{cases} u(0, t) = 0, \\ u(1, t) = 0, \\ u(x, 0) = \sin(2\pi x), \quad 0 \le x \le \le 1. \end{cases}

$$
