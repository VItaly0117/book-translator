# Prompt For Claude Pass 1

Прикрепи к сообщению **только** файлы из этой папки и используй этот текст как промпт.

---

Нужно выполнить **монтаж первой очереди** книги, без творческой переписи и без исправления всей книги.

Работай только по приложенным файлам.

Цель:
- взять рабочий markdown;
- вставить в него недостающие лекции из донорских extract-файлов;
- сохранить старый конфликтующий текст, не удаляя его;
- выдать новый markdown-файл как результат.

Что приложено:
- `pass1_donor_15_17.md`
- `pass1_donor_20.md`
- `pass1_donor_23.md`
- `pass1_current_anchor_14_18.md`
- `pass1_current_anchor_19_21.md`
- `pass1_current_anchor_22_24.md`

Что нужно сделать:

1. Взять за основу текущий рабочий текст книги.
2. Создать новый результат с названием:
   `current_working_v24_recovered_pass1.md`
3. Выполнить **только** такие вставки:
   - лекция 15
   - лекция 16
   - лекция 17
   - лекция 20
   - лекция 23

Куда вставлять:
- лекции 15–17 вставить в промежуток между лекцией 14 и лекцией 18
- лекцию 20 вставить между лекцией 19 и лекцией 21
- лекцию 23 вставить между лекцией 22 и лекцией 24

Правила:
- не переписывать остальную книгу;
- не менять стиль всей книги;
- не чинить LaTeX во всей книге;
- не удалять конфликтующий старый текст;
- если старый кусок явно конфликтует со вставленной лекцией, перенеси его вниз файла под заголовок:
  `## Невизначені фрагменти після монтажу`
- ничего не выкидывай безвозвратно.

Правило заголовков:
- каждая вставленная лекция должна иметь явный заголовок вида:
  `# Лекція N. ...`
- если в доноре русский заголовок, переведи **только заголовок** на украинский по смыслу;
- сам основной текст лекции можно пока оставить как есть.

Чего не делать:
- не трогать лекцию 28;
- не вставлять лекции 25, 29–35;
- не делать полную реконструкцию книги;
- не делать «cleanup» всей рукописи.

Что вернуть:
1. Только готовый markdown `current_working_v24_recovered_pass1.md`
2. После markdown коротко перечислить:
   - какие 5 лекций вставлены
   - какие старые фрагменты перенесены в `Невизначені фрагменти після монтажу`

Если видишь неоднозначность:
- не фантазируй;
- оставь метку `TODO: перевірити по оригіналу`;
- но текст сохраняй.

---

Если модель начинает тупить или пытаться переписывать всё:
- остановись;
- делай только монтаж по этим пяти лекциям.
