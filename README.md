# 📚 Azure Book Translator

Автоматизированный скрипт для перевода технической литературы (математика/физика) на украинский язык через Azure Translator, с полным сохранением LaTeX-формул и Markdown-разметки.

## ✨ Главные фичи
- **Нарезка тяжелых PDF на чанки:** Разбиение исходного PDF по 50 страниц через `pypdf` для обхода ограничений памяти и защиты от галлюцинаций OCR.
- **Интеллектуальное маскирование:** Формулы, уравнения и изображения маскируются токенами перед отправкой в Azure API, чтобы переводчик не сломал синтаксис LaTeX.
- **Жесткие абсолютные пути:** Для рендера Pandoc на Windows используются строгие абсолютные пути, чтобы изображения не терялись при генерации финальных PDF/EPUB.
- **Интерактивность:** Пользовательское CLI с поддержкой Drag & Drop, которое автоматически очищает кавычки Windows и предлагает удобные варианты.

## ⚙️ Установка и настройка (Windows)

1. **Установите стабильную версию Python 3.11 или 3.12.**

2. **Создайте виртуальную среду:**
   ```bash
   python -m venv venv
   ```

3. **Активируйте её:**
   ```bash
   .\venv\Scripts\activate
   ```

4. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Настройте ключи API:**
   Переименуйте файл `.env.template` в `.env` и впишите ключи:
   - `AZURE_TRANSLATOR_KEY`
   - `AZURE_TRANSLATOR_ENDPOINT`
   - `AZURE_TRANSLATOR_REGION`

## 🚀 Использование

Просто запустите скрипт из корня проекта:
```bash
python book_translator.py
```

Скрипт сам найдет исходники в папке `input/` или попросит перетащить файл в консоль. Все результаты сохраняются в папке `Output_Final/`!
